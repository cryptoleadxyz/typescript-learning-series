prerequisite
- install nodejs (nodejs.org)
- install typescript compiler (this compiles typescript to regular javascript for web deployment) - "npm install -g typescript"
- install "live server" extension on vscode
- to show console log -> right click on index.html -> select Open with live server

workflow
- to automatically compile continuously - "tsc [filename.ts] -w"
